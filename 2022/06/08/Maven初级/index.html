（1）scope标签

1.设定依赖使用范围，范围包括：

  • 主程序范围内有效（main文件夹范围内）
  • 测试程序范围内有效（test文件夹范围内）
  • 是否参与打包（pacakage指令范围内）

二.项目构建

根据插件来实现对项目生命周期的操作

  • mvn compile 编译
  • mvn clean 清理
  • mvn test 测试
  • mvn package 打包
  • mvn install 安装到本地仓库
  • mvn depoly 部署到私服

插件从上至下执行，和生命周期相关

除了生命周期插件，在pom文件中还可以引入其他插件，比如maven打包插件，tomcat插件等
